---
title: yieldless/resource
description: Native async disposal with acquire and release functions.
---

`yieldless/resource` turns a pair of acquire/release functions into an object
that participates in native `await using` cleanup.

## Exports

<div class="yl-signature"><code>type ResourceAcquire&lt;T&gt; = () =&gt; PromiseLike&lt;T&gt; | T</code></div>
<div class="yl-signature"><code>type ResourceRelease&lt;T&gt; = (resource: T) =&gt; PromiseLike&lt;void&gt; | void</code></div>
<div class="yl-signature"><code>interface AsyncResource&lt;T&gt; extends AsyncDisposable &#123; readonly value: T &#125;</code></div>
<div class="yl-signature"><code>acquireResource(acquire, release): Promise&lt;AsyncResource&lt;T&gt;&gt;</code></div>

## Example

```ts
import { acquireResource } from "yieldless/resource";

{
  await using db = await acquireResource(connect, disconnect);
  await db.value.query("select 1");
}
```

The release function runs once when the lexical scope exits, even if the block
throws.

## Where it fits

- Database or queue connections scoped to a request or job
- Temporary filesystem handles
- External clients that need explicit teardown

## Important detail

The resource wrapper exposes the underlying value as `.value`. That keeps the
disposable handle explicit and avoids pretending that current `await using`
syntax can destructure directly into a tuple.
